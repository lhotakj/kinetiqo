# Kinetiqo

Simple containerized Strava sync tool to database.

## Documentation

Visit our comprehensive documentation at [kinetiqo.lhotak.net](https://kinetiqo.lhotak.net)

Or view the documentation locally by opening `docs/index.html` in your browser.

## Quick Start

```bash
# Pull the latest image
docker pull kinetiqo/kinetiqo:latest

# Run with your configuration
docker run --env-file .env kinetiqo/kinetiqo:latest
```

## Features

- 🐳 **Containerized** - Deploy anywhere with Docker
- ⚡ **Fast & Efficient** - Optimized sync process
- 🔒 **Secure** - OAuth 2.0 authentication with Strava API
- 🔧 **Configurable** - Flexible database backend support
